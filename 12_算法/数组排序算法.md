## 选择排序法
### 【普通选择排序】
> [!基本思想]
> 1、每次趟结束后，将待排序数组的最小值移动到最左边。即每趟找小值。
> 2、 双层循环。外层循环 0...n-1。内层循环 i+1..n-1。
> 3、每次内层循环结束，通过比较，确认未排序区间的最小值（最大值）。
> 4、外层循环结束后，排序结束。

时间复杂度：O(n^2)。
```java
public int[] sortArray(int[] nums) {
    // 普通选择排序
    int n = nums.length;
    int min_index = 0;
    for (int i = 0; i < n - 1; i++) {
        min_index = i;
        for (int j = i + 1; j < n; j++) {
            if (nums[j] < nums[min_index]) {
                min_index = j;
            }
        }
        if (min_index != i) {
            int temp = nums[i];
            nums[i] = nums[min_index];
            nums[min_index] = temp;
        }
    }
    return nums;
}
```
### 【进阶】
#### 双向选择排序
每趟确定最小值和最大值。
注意：先确定的最小值，是否就是目标最大值。注意处理交换时的索引。

```java
public int[] sortArray(int[] nums) {
        // 双向选择排序
        int n = nums.length;
        int left = 0;
        int right = n - 1;
        int min_index = 0;
        int max_index = 0;
        while (left < right) {
            min_index = left;
            max_index = right;
            for (int i = left; i < right + 1; i++) {
                if (nums[i] < nums[min_index]) {
                    min_index = i;
                }
                if (nums[i] > nums[max_index]) {
                    max_index = i;
                }
            }

            int temp = nums[left];
            nums[left] = nums[min_index];
            nums[min_index] = temp;

            if (max_index == left) {
                max_index = min_index;
            }

            temp = nums[right];
            nums[right] = nums[max_index];
            nums[max_index] = temp;

            left++;
            right--;
        }
        return nums;        
    }
```

```java
class Solution {
	public int[] sortArray(int[] nums) {
		// 堆排序
		// 1、建堆，从最后一个非叶子节点开始。节点数 / 2 - 1。
		// 最大堆
		int n = nums.length;
		
		for (int i = n / 2 - 1; i > -1; i--) {
			heapify(nums, n, i);
		}
	
		for (int i = n - 1; i > 0; i--) {
			// 交换堆顶和当前末尾
			int temp = nums[i];
			nums[i] = nums[0];
			nums[0] = temp;
			// 堆缩小后的堆（大小 i，从堆顶下沉）
			heapify(nums, i, 0);
		}
		return nums;
	}
  
	public int[] heapify(int[] nums, int n, int i) {
		int largest = i;
		// 节点的左节点索引等于节点索引 * 2 + 1
		// 节点的右节点索引等于节点索引 * 2 + 2
		int left = 2 * i + 1;
		int right = 2 * i + 2;
		if (left < n && nums[left] > nums[largest]) {
			largest = left;
		}
	
		if (right < n && nums[right] > nums[largest]) {
			largest = right;
		}
	
		if (largest != i) {
			int temp = nums[i];
			nums[i] = nums[largest];
			nums[largest] = temp;
			heapify(nums, n, largest);
		}
		return nums;
	}
}
```
找最值。建堆是个好方法。可以将时间复杂度从O(n)降低到O(logn)。

##  插入排序法

> [!NOTE] 思想
> 1、将给定数组在逻辑上看做两部分。一部分是有序数组，一部分是待排序数组。有序范围从最左边索引为 0 的元素开始，一直向右扩张。
> 2、每趟从待排序数组取最左边一个元素，将其遍历的与有序数组元素比较，从有序数组的右边开始比较，如果待排序元素小于当前元素，则将当前元素向后移动一位。否则，就将待排序元素插入在比较元素之后。

可以通过嵌套两层for循环。也可外层for循环，内层while循环。
示例代码如下：
```java
public int[] sortArray(int[] nums) {
	// 插入排序
	int n = nums.length;
	if (n <= 1) return nums;
	for (int i = 1; i < n; i ++) {
		int temp = nums[i];
		int j;
		// j = i - 1;
		// while (j >= 0 && nums[j] > temp){
			// nums[j + 1] = nums[j];
			// j--;
		// }
		for (j = i - 1; j >= 0 && nums[j] > temp; j--) {
			nums[j + 1] = nums[j];
		}
		nums[j + 1] = temp;
	}
	return nums;
}
```
时间复杂度：O(n^2)。
空间复杂度：原地排序O(1)。
稳定排序。

适合规模小的数组、基本排序的数组。
