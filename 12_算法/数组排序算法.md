### 选择排序法
#### 【普通选择排序】
双层循环。


> [!NOTE]基本思想
> 1、每次趟结束后，将待排序数组的最小值移动到最左边。即每趟找小值。
> 2、外层循环 0...n-1。
内层循环 i+1..n-1。
> 2、每次内层循环结束，通过比较，确认未排序区间的最小值（最大值）。

外层循环结束后，排序结束。
时间复杂度：O(n^2)。
```java
public int[] sortArray(int[] nums) {
    // 普通选择排序
    int n = nums.length;
    int min_index = 0;
    for (int i = 0; i < n - 1; i++) {
        min_index = i;
        for (int j = i + 1; j < n; j++) {
            if (nums[j] < nums[min_index]) {
                min_index = j;
            }
        }
        if (min_index != i) {
            int temp = nums[i];
            nums[i] = nums[min_index];
            nums[min_index] = temp;
        }
    }
    return nums;
}

找最值。建堆是个好方法。可以将时间复杂度从O(n)降低到O(logn)。
```java
class Solution {
	public int[] sortArray(int[] nums) {
		// 堆排序
		// 1、建堆，从最后一个非叶子节点开始。节点数 / 2 - 1。
		// 最大堆
		int n = nums.length;
		
		for (int i = n / 2 - 1; i > -1; i--) {
			heapify(nums, n, i);
		}
	
		for (int i = n - 1; i > 0; i--) {
			// 交换堆顶和当前末尾
			int temp = nums[i];
			nums[i] = nums[0];
			nums[0] = temp;
			// 堆缩小后的堆（大小 i，从堆顶下沉）
			heapify(nums, i, 0);
		}
		return nums;
	}
  
	public int[] heapify(int[] nums, int n, int i) {
		int largest = i;
		// 节点的左节点索引等于节点索引 * 2 + 1
		// 节点的右节点索引等于节点索引 * 2 + 2
		int left = 2 * i + 1;
		int right = 2 * i + 2;
		if (left < n && nums[left] > nums[largest]) {
			largest = left;
		}
	
		if (right < n && nums[right] > nums[largest]) {
			largest = right;
		}
	
		if (largest != i) {
			int temp = nums[i];
			nums[i] = nums[largest];
			nums[largest] = temp;
			heapify(nums, n, largest);
		}
		return nums;
	}
}
