
> [!NOTE] 思想
> 1、将给定数组在逻辑上看做两部分。一部分是有序数组，一部分是待排序数组。有序范围从最左边索引为 0 的元素开始，一直向右扩张。
> 2、每趟从待排序数组取最左边一个元素，将其遍历的与有序数组元素比较，从有序数组的右边开始比较，如果待排序元素小于当前元素，则将当前元素向后移动一位。否则，就将待排序元素插入在比较元素之后。

可以通过嵌套两层for循环。也可外层for循环，内层while循环。
示例代码如下：
```java
public int[] sortArray(int[] nums) {
	// 插入排序
	int n = nums.length;
	if (n <= 1) return nums;
	for (int i = 1; i < n; i ++) {
		int temp = nums[i];
		int j;
		// j = i - 1;
		// while (j >= 0 && nums[j] > temp){
			// nums[j + 1] = nums[j];
			// j--;
		// }
		for (j = i - 1; j >= 0 && nums[j] > temp; j--) {
			nums[j + 1] = nums[j];
		}
		nums[j + 1] = temp;
	}
	return nums;
}
```
时间复杂度：O(n^2)。
空间复杂度：原地排序O(1)。
稳定排序。

适合规模小的数组、基本排序的数组。
