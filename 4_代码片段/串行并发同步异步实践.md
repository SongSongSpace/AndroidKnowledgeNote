# 串行并发同步异步实践-260417

## runBlocking (同步-串行)
```kotlin
fun syncSerial() {
    val start = System.currentTimeMillis()
    
    // 同步执行：每个任务完成后才执行下一个
    // 体现：阻塞等待，无并发
    runBlocking {
        task("A").let { println(it) }
        task("B").let { println(it) }
        task("C").let { println(it) }
    }
    
    val end = System.currentTimeMillis()
    println("同步串行耗时: ${end - start}ms") // ~600ms
}
```

 ## 单线程池（异步提交、同步等待获取返回值Future、单线程=串行）
 ```kotlin
fun asyncSerial() {
    val executor = Executors.newSingleThreadExecutor() // 单线程 = 串行
    val start = System.currentTimeMillis()
    
    // 异步提交：主线程不等待，但任务在单线程池中串行执行
    // 体现异步：submit 立即返回 Future
    // 体现串行：单线程池一次只处理一个任务
    val future1 = executor.submit { runBlocking { task("A") } }
    val future2 = executor.submit { runBlocking { task("B") } }
    val future3 = executor.submit { runBlocking { task("C") } }
    
    // 同步等待：阻塞主线程收集结果（将异步转为同步）
    println(future1.get())
    println(future2.get())
    println(future3.get())
    
    executor.shutdown()
    val end = System.currentTimeMillis()
    println("异步串行耗时: ${end - start}ms") // ~600ms
}
```

## 多线程线程池（异步提交、并发执行任务）
```kotlin
// 3. 异步：线程池 + 并发
fun asyncConcurrent() {
    val executor = Executors.newFixedThreadPool(3) // 3个线程 = 并发
    val start = System.currentTimeMillis()
    
    // 异步提交：三个任务同时被三个线程执行
    // 体现并发：任务时间线重叠
    val futures = listOf("A", "B", "C").map { name ->
        executor.submit { runBlocking { task(name) } }
    }
    
    // 同步等待所有完成
    futures.forEach { println(it.get()) }
    
    executor.shutdown()
    val end = System.currentTimeMillis()
    println("异步并发耗时: ${end - start}ms") // ~200ms
}
```

## 结构并发
```kotlin
suspend fun coroutineConcurrent() = coroutineScope {
    val start = System.currentTimeMillis()
    
    // 体现协程：挂起函数，非阻塞
    // 体现并发：三个 async 同时启动
    val deferreds = listOf("A", "B", "C").map { name ->
        async { task(name) } // async 立即执行，不等待
    }
    
    // 等待所有完成
    deferreds.awaitAll().forEach { println(it) }
    
    val end = System.currentTimeMillis()
    println("协程并发耗时: ${end - start}ms") // ~200ms
}
```

## 普通串行执行
```kotlin
suspend fun coroutineSerial() = coroutineScope {
    val start = System.currentTimeMillis()
    
    // 体现串行：逐个调用，无 async 或 launch 包装
    println(task("A"))
    println(task("B"))
    println(task("C"))
    
    val end = System.currentTimeMillis()
    println("协程串行耗时: ${end - start}ms") // ~600ms
}
```

|对比项|情况|
|---|---|
|异步|调用方不阻塞线程等待|
|同步|调用方阻塞线程等待|
|并发|不同的线程在同一段时间一起在执行任务|
|串行|一个线程执行完一个任务才能执行下一个任务|
